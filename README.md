# 🍞 서비스 목표
- **배경 및 목표**
  - 동네 빵집의 갓 나온 따뜻한 빵, 신선한 빵을 소비자에게 제공하고 가게 매출증대에 도움이 되는 선순환 구조를 가진 서비스


<br/>

# 🍞 서비스 특징
- **고객님**
  - 소셜 로그인 및 알림 서비를 활용하여 교객의 편의성 증대
  - 편리한 주문 및 배달 시스템을 제공하여 온라인 주문을 촉진
- **사장님**
  - 판매량 추적을 통해 사장님은 영업 성과를 한눈에 확인
  - 프로젝트를 통해 사용자들이 지역 빵집을 쉽게 찾아서 방문할 수 있도록 함으로써 지역 사회와 소통 및 상호 연결 강화
